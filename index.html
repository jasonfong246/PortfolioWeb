Endless Runner

This Endless Runner game, inspired by Subway Surfers and Temple Run, was developed by a two-person team using Unity, C# scripting, and the Unity Input System for precise player movement control. The player automatically advances through an endless environment while avoiding obstacles that spawn in randomized lanes (left, middle, or right) to ensure fast-paced and unpredictable gameplay. The game includes a dynamic difficulty system where the player’s running speed gradually increases over time, creating a more challenging experience as the run progresses. To maintain performance, objects that the player has passed are efficiently removed from the scene, preventing unnecessary memory buildup and ensuring smooth gameplay. This project showcases our understanding of Unity’s component-based architecture, object pooling, event-driven movement controls, and real-time procedural obstacle generation.

Eat Together Application

Eat Together is an Android application developed using Android Studio and tested via its built-in emulator. Prey & Predator Simulation

This project simulates the dynamic interaction between prey (blue) and predators (red), each with their own movement patterns and vision logic, similar to how real animals like birds and cats perceive their environment. The prey attempts to navigate the map while avoiding obstacles, including a moving pillar that blocks escape paths.

Predators roam freely until they collide with a wall, at which point they change direction. When a prey enters a predator’s field of vision, the predator will actively chase it until it moves out of sight. The behavior is controlled through custom C# scripts in Unity, implementing AI states, detection ranges, and real-time decision-making.

Game of Set

The Game of Set is a fast-paced pattern-matching card game where each card contains a unique combination of four attributes; number, shape, color, and shading. A “Set” is formed when, for every attribute, the three selected cards are either all the same or all different. I built the full game using HTML, CSS, and JavaScript, including card rendering, board layout, and interactive UI elements for card selection. The JavaScript logic handles real-time pattern validation, checking whether the chosen cards satisfy Set rules using attribute comparison functions. The project also includes automatic Set generation, deck shuffling, and a dynamic UI that highlights matches, handles invalid selections, and updates the board seamlessly. The result is a complete, functioning browser version of the classic Set game with clean design, responsive interactions, and accurate rule enforcement.

Maze Game

The Maze Generator is an interactive Unity game that creates randomized mazes each time it runs, providing a fresh challenge for players with every playthrough. Players navigate the maze to collect coins and earn points, adding an extra layer of engagement and replayability. The maze generation algorithm uses Kruskal’s algorithm to build the structure and generate dead-ends, ensuring a unique layout every time. The project also includes visualization of the maze generation process, giving players a dynamic view of the maze as it is created. Developed with Unity and scripted in C#, this project highlights skills in algorithm implementation, procedural content generation, game mechanics, and real-time visualization, while delivering an interactive and engaging gameplay experience.
